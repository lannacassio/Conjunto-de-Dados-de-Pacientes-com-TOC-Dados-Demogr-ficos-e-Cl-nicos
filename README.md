# Conjunto-de-Dados-de-Pacientes-com-TOC-Dados-Demogr-ficos-e-Cl-nicos
É uma coleção abrangente de informações referentes a 1500 indivíduos diagnosticados com Transtorno Obsessivo-Compulsivo (TOC). Este conjunto de dados abrange uma ampla gama de parâmetros, fornecendo uma visão detalhada dos perfis demográficos e clínicos desses indivíduos.
